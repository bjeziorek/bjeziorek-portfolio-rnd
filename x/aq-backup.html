1. CIR — Cognitive Regulation Model (5‑State Framework)

CIR (Continuum · Introspection · Regulation) is a mechanistic model describing how cognitive regulation shifts across five states:

  • CIR‑1 Breakdown: Severe dysregulation, reactive behaviour — tests AI safety boundaries, tone control, escalation risks.

  • CIR‑2 Collapse: High dysregulation, fragile control — tests clarity, emotional load, misinterpretation risks.

  • CIR‑3 Fragile Stability: Functioning with high cognitive cost — tests complexity, pacing, cognitive load.

  • CIR‑4 Stable Regulation: Baseline stability — tests default behaviour and consistency.

  • CIR‑5 Enhanced Stability: Peak performance — tests advanced reasoning, adaptability, creativity.

2. FEDRI — Behavioural Simulation & Forensics Pipeline

FEDRI is an 8‑stage pipeline for modelling perception, reaction, and emergent behaviour in complex systems.

  • Initialize Agent — goals, fears, constraints, CIR baseline

  • Initialize Scene — environment, stakes, timing

  • Perception Pass — subjective interpretation

  • Reaction Pass — emotion, impulse, action

  • Stimulus Injection — new information or stressor

  • Multi‑Agent Loop — dynamics evolve, CIR shifts

  • Distillation Pass — extract behavioural signatures

  • Integration Pass — update agent model
